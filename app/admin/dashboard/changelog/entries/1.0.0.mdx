## Added

- Initial release of the Ishimwe Jean Baptiste website.
- Fully responsive design optimized for all devices.
- Integrated homepage featuring a hero section, about section, and program highlights.
- Public pages including: About, Programs, Blogs, Team, Impact, Gallery, Contact Us, FAQs, Privacy Policy, and Terms.
- Admin panel with features such as dark/light mode, notifications, search functionality, and a collapsible sidebar.
- Blog and gallery management integrated via Hygraph.
- Email notifications for messages and subscriptions.
- Enhanced security features including two-factor authentication (2FA) and updated password policies.
- Performance tracking using Google Analytics, Sentry, and Better Stack.
